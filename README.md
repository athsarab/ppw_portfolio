# IT4070 Professional Portfolio

Single-page academic and professional portfolio built with Next.js, Tailwind CSS, and GSAP. The site is designed for both web viewing and print-ready PDF export.

## Key Features

- Single-page layout with smooth in-page navigation
- Glassmorphism cards, bento layouts, and academic-ready typography
- GSAP motion with reduced-motion and print-safe fallbacks
- Print-optimized PDF output with page breaks and clean styling

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Lint

```bash
npm run lint
```

## Print to PDF

Use the "Print / Save as PDF" button in the UI, or use the browser print dialog.
